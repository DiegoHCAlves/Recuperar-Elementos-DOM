Como Posso encontrar elementos

  • Pela Tag

    Usando o "document.getElementsByTagName()"

    Ele retorna um array com todos elementos que possuem o nome de tag citado.

    Por exemplo:

    Se você clicar no botão abaixo: o Java será pintado de azul, o fundo do Python de vermelho e o NodeJs virará SpringBoot



    • Java
    • Python
    • NodeJs

    Sintaxe:

    element = document.getElementsByTagName('li')[1]
    element.style.color = 'blue';

    element = document.getElementsByTagName('li')[2]
    element.style.background = 'red';

    element = document.getElementsByTagName('li')[3]
    element.innerText = "SpringBoot";

  • Pelo Id

    Usando o "document.getElementById()"

    Ele retorna o elemento pelo id citado.

    Por exemplo:

    Se você clicar no botão abaixo: o Java será pintado de azul, o fundo do Python de vermelho e o NodeJs virará SpringBoot



    • Java
    • Python
    • NodeJs

    Sintaxe:

    element = document.getElementById("java")
    element.style.color = 'blue';

    element = document.getElementById("python");
    element.style.background = 'red';

    element = document.getElementById("nodejs");
    element.innerText = "SpringBoot";

  • Pelo Name

    Usando o "document.getElementByName()"

    Ele retorna um Array com os elementos com aquele nome, sendo necessário indicar a posição, assim como no getElementsByTagName.

    Por exemplo:

    Se você clicar no botão abaixo: o Java será pintado de azul, o fundo do Python de vermelho e o NodeJs virará SpringBoot



    • Java
    • Python
    • NodeJs

    Sintaxe:

    element = document.getElementsByName("java")[0]
    element.style.color = 'blue';

    element = document.getElementsByName("python")[0]
    element.style.background = 'red';

    element = document.getElementsByName("nodejs")[0]
    element.innerText = "SpringBoot";

  • Pela Classe

    Usando o "document.getElementsByClassName()"

    Ele retorna um Array com os elementos pertencentes aquela classe, sendo necessário indicar a posição, assim como no getElementsByTagName e o getElementsByName.

    Por exemplo:

    Se você clicar no botão abaixo: o Java será pintado de azul, o fundo do Python de vermelho e o NodeJs virará SpringBoot



    • Java
    • Python
    • NodeJs

    Sintaxe:

    element = document.getElementsByClassName('langs')[0]
    element.style.color = 'blue';

    element = document.getElementsByClassName('langs')[1]
    element.style.background = 'red';

    element = document.getElementsByClassName('langs')[2]
    element.innerText = "SpringBoot";

  • Pelo seletor do CSS

    Usando o "document.querySelectorAll("--caminho--")"

    Ele retorna um array dos elementos que pertencem ao caminho indicado. Este em especial tem potencial para ser explorado a fundo podendo chamar os elementos pelo #id ou .class e assim sucessivamente.

    Por exemplo:

    Se você clicar no botão abaixo: o Java será pintado de azul, o fundo do Python de vermelho e o NodeJs virará SpringBoot



    • Java
    • Python
    • NodeJs

    Sintaxe:

    let element = document.querySelectorAll('.exSelector');

    element[0].style.color = 'blue';
    element[1].style.background = 'red';
    element[2].innerText = "SpringBoot";
